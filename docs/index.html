GitHub Pages site for the Security Advisor project

Run scans, generate SARIF, and publish findings into Vulnerability Manager.

Security Advisor is an MCP server that coordinates SAST, SCA, IaC, and container security checks, merges the output into a single SARIF report, and can publish normalized findings into the Vulnerability Manager app for tracking by organization, project, service, and version.
